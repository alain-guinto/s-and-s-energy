## üöÄ Getting Started

Follow these instructions to get a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

*   **Node.js**: Ensure you have Node.js installed on your system.
*   **npm**: The Node package manager comes with Node.js.

### Installation

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/s-s-energy.git
    cd s-s-energy
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Configure Environment Variables:**
    Create a `.env.local` file in the root directory and add your Google Gemini API key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```
    *(Note: Ensure you obtain a valid API key from Google AI Studio)*

4.  **Run the application:**
    ```bash
    npm run dev
    ```
    This will start the development server, typically accessible at `http://localhost:5173`.
